在Github上搭建个人博客和网站

准备前提:
GitHub官网： https://github.com/
Github账号
加速器：这边建议使用镜像网站或者插件(没有也可以只是访问GitHub官网会比较慢)
Node.js 安装包，官网： https://nodejs.org/zh-cn/download/ （以有请忽略）
Git 安装包 官网： https://git-scm.com/downloads/ （以有请忽略）
（特别提醒：没有Github账号的读者自行注册，此处不再赘述）

一、建仓库、设置

1、在右上角点击个人头像找到”you respositories“点击

2、选择NEW

3、选择仓库名称一般为个人账号+github.io 例如：我这个账号名称为HEAVEN-T那么就该写：heaven-t.github.io（实际上是你可以把”个人账号“这一块替换成你喜欢的任意名称，前提是不能与站内的重复)
4、默认勾选Public不要动
5、Add .gitignore内选择  node  
（readme 可加可不加，后面建好后还能添加）

如果名字重复会出现以下情况（这个仓库已建好，再建就重复了）

6、操作完成点击下方的创建即可

这是一个新建好的仓库，里面啥都没有

二、安装Node.js

进入网站以后，根据自己的电脑系统，选择不同的平台下载。目前支持的主要平台是Windows,MacOS,Linux。下载的版本推荐是长期维护版，另外一个版本是最新尝鲜版，虽然包含了很多最新功能，但是有可能不太稳定，不太适合新人使用。

第一步：开始安装

1、下载完成后，双击“node-v11.5.0-x64.msi”，开始安装Node.js

第二步：点击【Next】按钮

第三步：勾选复选框，点击【Next】按钮。

第四步：修改好目录后，点击【Next】按钮。

第五步：安装完后点击【Finish】按钮完成安装。

第六步：测试。

1、至此Node.js已经安装完成，可以先进行下简单的测试安装是否成功了，后面还要进行环境配置

2、在键盘按下【win+R】键，输入cmd，然后回车，打开cmd窗口

3、安装完后的目录如下图所示

4、此处说明下：新版的Node.js已自带npm，安装Node.js时会一起安装，npm的作用就是对Node.js依赖的包进行管理，也可以理解为用来安装/卸载Node.js需要装的东西

三、安装Git

1. Git 的下载

打开官网下载适合自己电脑的版本，选择【Download】下载

2. Git 安装

下载后双击Git的安装包开始安装，基本上每一步都是傻瓜式安装
点击【Next】下一步

接下来选择路径（一般不选择C盘）

下面页面大家根据自己需要选择勾选即可，不过一般都是默认的

这里默认即可
这是在 “开始” 菜单文件夹中创建程序的快捷方式

Git 安装程序内有很多编辑器，我们选择默认的即可
如果选其他编辑器，则需要去其官网安装后才能进行下一步

接下来基本全是默认的，直接【Next】即可

截图可能不是那么全，你只需要知道一直下一步即可

然后选择【Install】安装

安装好后用鼠标在桌面空白处右击会出现Git Bash Here 和Git GUI Here 的图标

点击Git Bash Here图标输入：
where git ：查看安装位置
git –version ：查看安装是否成功

Git 配置

右键”此电脑”→”属性”→”高级系统设置”

选择环境变量

在系统变量内选择Path，点击编辑

加入刚才的安装位置
