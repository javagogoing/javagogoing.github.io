循环

什么是循环

循环是程序设计语言中反复执行某些代码的一种计算机处理过程，常见的有按照次数循环和按照条件循环。

三种循环的总结

While

while循环语法
    循环变量初始化
    while（循环条件）{
        循环体操作（循环中重复执行的操作）
        控制循环语句（循环变量的改变）
    }
    
while死循环语法
    while(true){
        //循环体
        break;
    }
    
特点：
    先判断，再执行

do…white

do...while（）循环
    循环的初始化值
    do{
        循环体
        控制循环语句
    }while（循环条件）；

do...while;死循环语法
    do{
        //执行体
        break;
    }while(true);

特点:
    while先执行，后判断

do...while（）和while的区别
    while：先判断，后执行
    do..while：先执行，后判断
    初始情况不满足循环条件时
        while循环一次都不会执行
        do-while循环不管任何情况都至少执行一次
        
break语句
    满足特定条件时便执行break语句跳出循环
continue语句
    用于结束本次循环，进入下一次循环，并不是跳出循环

    for(int i=0 ; i<=12 ; i++){
        //循环体
    }
    for(初始化；条件；迭代){
        //循环体
    }
for死循环语法
    for（；；；）{
        //循环体
    }

三种循环的总结

学习了Java提供的三种最主要的循环结构，他们是while、do-while和for循环结构。无论哪一种循环结构，都有四个必不可少的部分：初始化、循环条件、循环体、迭代部分，缺少了任何一个都可能造成严重的错误。

语法不同

whlie循环语句
    while(<条件>){
        //循环体
    }
do-while循环语句
    do{
        //循环体
    }while(<循环条件>);
    
for循环语句
    for(初始化；条件；迭代){
        //循环体
    }

执行顺序不同

while循环
    先进行条件判断，再执行循环体，如果条件不成立，退出循环
do-while循环
    先执行循环体，再进行条件判断，循环体至少执行一次。
for循环
    先执行初始部分，再进行条件判断，然后执行循环体，最后进行迭代部分的计算，如果条件不成立，跳出循环。

适合情况不同

在解决问题时，对循环次数确定的情况，通常选用for循环。
对循环次数不确定的情况，通常选用while和do+while循环。

数组

为什么使用数组

遇到多个相同类型的数据时，为了方便使用和调用就需要用到数组来储存这些数据，

什么是数组

数组是一个变量，储存相同数据类型的一组数据
数组就像一列火车，而数组的每个元素就是这火车的每一节车厢。每个元素都可以当作单个变量来使用。而每一个元素必须是相同类型。
数组的下标（索引）:0相对应于第1个元素 ，1相对应于第2个元素。

数组和变量的区别

声明一个变量就是在内存空间划出一块的空间
声明一数组就是在内存空间划出划出一连串的空间

    元素类型

在不同的方法中·，可以有同名局部变量
在同一个类中，成员变量和局部变量同名时，局部变量具有更高的优先级

打开帮助文档
点击显示，索引，看到输入框
你要学习什么内容，你就在框框里输入什么内容
比如：Random

看包

java.lang包下的类在使用的时候是不需要导包的

看类的表述

Random类是用于生成随机数的类

看构造方法

Random( ):无参构造方法
    Random r = Random();

看成员方法

public int nextInt(int n);产生的是一个[ 0，n ]范围内的随机数

调用方法

看返回值类型：人家返回什么类型，你就用什么类型接收
看方法名:名字不要写错了
看形式参数：人家要几个参数，你就用几个参数，你就给几个，人家要什么数据类型的，你就就给什么数据类型的
    int number = r.nextInt(100);

字符串操作

String类

创建对象
    String s="Hello World";
    String s=new String();
    String s=new String("Hello Word");

常用方法
    获得字符串的长度： toString()
    比较字符串： equals()
    连接字符串： concat()
    提取字符串：  substring()
    搜索字符串： indexOf()
    拆分字符串： split

StringBuffer类

创建对象
    StringBuffer sb=new StringBuffer();
    StringBuffer sb=new StringBuffer("bdqn");

常用方法
    转换成String类型： toString()
    连接字符串： append()
    插入字符串：insert()

String常用方法

String：引用数据类型（有属性和方法）：java.lang包，这个包底下的类不需要自我介绍

获取字符串长度：length();
    无参有返回
    参数：无
    返回： int
    
比较字符串长度：equals（需要比较的字符串）---区分大小写
    有参有返回
    参数：String  
    返回：boolean
    
比较字符串：equalslgnoreCase()--不区分大小写
    有参有返回
    参数：String
    返回：boolean

小写转换成大写：:toUpperCase()
    无参有返
    返回： String
    
大写转换成小写：  toLowerCase()
    无参有返
    返回：String
    
通过字符串查找下标： indexOf()--从前往后
    有参有返回
    参数
        String    indexOf("需要查找的字符串")；
        String int indexOf("需要查找的字符串",查找字符串);
    返回
        int （下标位置，如果没有找到返回-1）
        
通过字符串找下标： lastIndexOf()--从后往前
    有参有返回
    参数
        String lastIndexOf(“需要查找的字符串”)；
        Sting int lastIndexOf("需要查找的字符串"，查找的起始位置);
    返回
        int （下标位置，如果没有返回-1）
        
去除两边的空格： trim();
    无参有返
    返回：String
    
替换字符串：replace()
    有返有返
    参数：String,String
    返回值：String
    使用：String  新的字符串=replace（“需要替换的字符串”，“替换后的字符串”）；
    
切割：split()
    有参有返回
    参数： String
    返回： String[ ]
    
拼接： concat（）--和+效果一样
    有参有返回
    参数：String
    返回： String

截取 substring()
    有参有返回
    参数
        int substring(截取的起始位置);
        int int substring(截取的起始位置，截取的结束位置）--包前不包后
    返回：String
    
拆分字符串： charAt()
    有参有返回
    参数： String
    返回：char
