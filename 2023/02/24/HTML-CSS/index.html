定位属性

position

static: 默认值，没有定位
absolute:绝对定位 参考点：document(网页文档)
relative:相对定位 参考点：原本自身的位置
fixed:固定定位 参考点：浏览器窗口

设置的定位之后，搭配left、top、right、bottom做位置的调整

溢出属性

overflow:hidden/scroll
hidden: 内容溢出部分隐藏
scroll: 内容溢出自动加滚动条

position 属性值的含义

static元素框正常生成。块级元素生成一个矩形框，作为文档流的一部分，行内元素则会创建一个或多个行框，置于其父元素中。

relative元素框偏移某个距离。元素仍保持其未定位前的形状，它原本所占的空间仍保留。

absolute元素框从文档流完全删除，并相对于其包含块定位。包含块可能是文档中的另一个元素或者是初始包含块。元素原先在正常文档流中所占的空间会关闭，就好像元素原来不存在一样。元素定位后生成一个块级框，而不论原来它在正常流中生成何种类型的框。fixed元素框的表现类似于将 position 设置为 absolute，不过其包含块是视窗本身。

提示：相对定位实际上被看作普通流定位模型的一部分，因为元素的位置相对于它在普通流中的位置

选择过渡属性

transition-property
例：只对背景和左外边距过渡transition-property:background,margin-left; 
对所有属性过渡transition-property: all;

过渡时间，单位为秒
    transition-duration:2s;
    
延时时间，单位为秒
    transition-delay:1s;
    
时间曲线
    transition-timing-function:ease-in;
      属性值：linear 以相同的时间开始和结束
               ease 慢速开始，然后变快，然后慢速结束
              ease-in 慢速开始
        ease-out 慢速结束

过渡属性的简写：
    transition: all 2s linear 0s;

阴影

box-shadow:x轴 y轴 模糊度 阴影大小
例：box-shadow: 0px 2px 10px 0px;

动画的实现

一步：声明一个动画

@keyframes aaa
    {
    /*0% 50% 100%都是关键帧，可自定义百分比*/
    0%{
        margin-left:0px
    }
    50%{
        margin-left:100px
    }
    100%{
        margin-left:0px
    }
    }

第二步：给需要应用动画的元素添加动画属性

指定动画名：animation-name:aaa;
指定动画时长：animation-duration:2s;
动画延时时间：animation-delay:0s;
动画播放次数：animation-iteration-count:n; //n可以自定义次数，infinite表示无限次

简写：
    animation: name duration  delay iteration-count;
    例：animation: aaa 2s 0s infinite;
